js-min-bench minifier comparison

This page compares JavaScript minifiers across various settings and for various inputs.

Note: the input programs are not comparable against one another because they do wildly different things. So don't compare "angularjs" vs "react", but rather for a given input program compare the different minifiers (uglify vs closure etc.).

See the README for more background on the methodology. Scroll past the results to see more details about what these numbers are.
